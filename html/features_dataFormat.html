CẤU TRÚC DỮ LIỆU SELECT2

Select2 có thể hiển thị dữ liệu được cung cấp theo chương trình từ một mảng hoặc nguồn dữ liệu từ xa (AJAX) dưới dạng các tùy chọn thả xuống. Để thực hiện điều này, Select2 yêu cầu một định dạng dữ liệu rất cụ thể.

Định dạng này bao gồm một đối tượng JSON chứa một mảng các đối tượng được khóa bằng khóa results. 

    {
      "result": [
         {
          "id": 1,
          "text": "Tùy chọn 1"
         },
         {
          "id": 2,
          "text": "Tùy chọn 2"
         }
      ],
      "pagination": {
          "more": true
         }
    }

Select2 yêu cầu mỗi đối tượng phải chứa một thuốc tính id và text. Các tham số bổ sung được truyền vào cùng với các đối tượng dữ liệu sẽ được bao gồm trong các đối tượng dữ liệu mà Select2 hiển thị.

Dữ liệu phản hồi cũng có thể chứa dữ liệu phân trang, nếu bạn muốn sử dụng tính năng "cuộn vô hạn". Dữ liệu này cần được chỉ định trong thuộc tính pagination.



TÍNH NĂNG CHỌN VÀ TẮT

Bạn cũng có thể cung cấp các thuộc tính selected và disabled cho các tùy chọn trong cấu trúc dữ liệu này. Ví dụ: 

    {
      "result": [
         {
          "id": 1,
          "text": "Tùy chọn 1"
         },
         {
          "id": 2,
          "text": "Tùy chọn 2",
          "selected": true
         }
         {
          "id": 3,
          "text": "Tùy chọn 3",
          "disable": true
         }
      ],
      "pagination": {
          "more": true
         }
    }


Ví dụ

CHỌN QUỐC GIA 
      var myData: [
         {
          "id": 'VN',
          "text": 'Việt Nam',
          "selected": true
         },
         {
          "id": 'US',
          "text": 'Hoa Kỳ',
          "disabled": true
         },
         {
          "id": 'JP',
          "text": 'Nhật Bản'
         },
         {
          "id": 'KR',
          "text": 'Hàn Quốc'
         },
      ]

    $("country-select-Select2").select2({
        data: myData,
        allowClear: true
    });


DỮ LIỆU NHÓM

Khi tạo tùy chọn trong phần <optgroup>, các tùy chọn nên được lồng vào khóa con (children) của mỗi đối tượng nhóm. Nhãn cho nhóm nên được chỉ định là thuộc tính text trên đối tượng dữ liệu tương ứng của nhóm. 

    {
      "result": [
         {
          "text": "Nhóm 1",
          "children": [
            {
                "id": 1,
                "text": "Tùy chọn 1.1"
            },
            {
                "id": 2,
                "text": "Tùy chọn 1.2"
            }
          ]
         },
         {
           "text": "Nhóm 2",
           "children": [
             {
                "id": 3,
                "text": "Tùy chọn 2.1"
             },
             {
                "id": 4,
                "text": "Tùy chọn 2.2"
             }
           ]
         }
       ] 
    }


Ví dụ

CHỌN QUỐC GIA 
     var myData1: [
         {
          "text": "Châu Á",
          "children": [
            {
                "id": "VN",
                "text": "Việt Nam"
            },
            {
                "id": "CHN",
                "text": "Trung Quốc"
            }
          ]
         },
         {
           "text": "Châu Âu",
           "children": [
             {
                "id": "FR,
                "text": "Pháp"
             },
             {
                "id": "ITA",
                "text": "Ý"
             }
           ]
         }
       ]

    $("country1-select-Select2").select2({
        data: myData1,
        allowClear: true
    });
CẢNH BÁO

Vì Select2 tạo ra <optgroup> khi tạo các tùy chọn lồng nhau, nên chỉ hỗ trợ một cấp độ lồng nhau duy nhất. Không đảm bảo bất kỳ cấp độ lồng nhau bổ sung nào sẽ được hiển thị chính xác trên tất cả các trình duyệt và thiết bị.
